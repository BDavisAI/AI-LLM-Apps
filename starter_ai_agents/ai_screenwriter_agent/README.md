# 🎬 AI Screenwriter Agent

An AI-powered helper for generating screenplay outlines, character profiles, and opening scenes. Built with **Streamlit** and **Agno** using OpenAI models.

## 🚀 Features
- Generate logline and story outline from a short idea.
- Produce character descriptions and opening scene dialogue.
- Simple Streamlit interface with API key input.

## 📦 Installation
```bash
pip install -r requirements.txt
```

## 🔑 Environment Variables
Set the following in the UI when running the app:
- `OPENAI_API_KEY`: OpenAI API key

## ▶️ Usage
```bash
streamlit run screenwriter_agent.py
```

Enter your story idea and click **"Generate Screenplay"** to receive a structured outline.
